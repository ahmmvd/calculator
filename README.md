Minimalistic Calculator
A small web-based calculator with a clean dark UI, built using only HTML, CSS, and vanilla JavaScript.
Features:
•	Basic arithmetic operations: addition, subtraction, multiplication, and division.
•	Percentage calculation with a dedicated % button.
•	Parentheses support for more complex expressions.
•	Clear (C) and undo (backspace) buttons to easily fix input mistakes.
•	Responsive, keyboard-like layout with a modern dark theme and subtle shadows.
Tech Stack:
HTML for structure and calculator layout.
CSS for styling, dark theme, and responsive grid of buttons.
JavaScript for calculator logic (input handling, evaluation, percent, undo, parentheses toggle).

You can try it through here: https://ahmmvd.github.io/calculator/
